# 📊 SQL Project 3 – Sales & Revenue Analysis (Northwind)

This project analyzes sales and revenue performance using the Northwind sample database.

The goal of this project is to apply intermediate SQL concepts to perform revenue-driven analysis that supports business decision-making.

The analysis explores how revenue is distributed across products, categories, time, and customers, and includes customer segmentation using advanced SQL logic.

---

## 🛠 Tools Used
- SQL Server
- Northwind Sample Database

---

## 🔍 Business Questions Answered
- Which products generate the highest revenue?
- Which product categories contribute most to total revenue?
- How does revenue change over time (monthly trends)?
- How can customers be segmented based on revenue contribution?

---

## 🧠 SQL Concepts Used
- Multi-table JOINs
- Revenue calculations using `SUM(Price × Quantity)`
- `GROUP BY` and `ORDER BY`
- Date functions (`YEAR`, `MONTH`)
- Common Table Expressions (CTEs)
- `CASE WHEN` for customer segmentation

---

## 📌 Key Takeaways
- Revenue is concentrated among a small number of products and customers
- Category-level analysis provides strong business insights
- Revenue shows clear seasonal patterns
- Customer segmentation helps identify high-value and growth-potential customers
