# SQL Customer Concentration Analysis

## Project Overview
This project analyzes customer distribution across countries using the Northwind database to identify geographic concentration.

## Business Question
Do a small number of countries account for a significant portion of customers?

## Dataset
- Database: Northwind
- Table: Customers

## Analysis Summary
Customers were grouped into Top Markets (USA, France, Germany) and Other Countries to evaluate customer concentration.

## Key Insight
Nearly 40% of customers come from just three countries, indicating strong geographic concentration.
